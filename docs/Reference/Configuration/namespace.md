# Maple.Json.ObjectAsPrimitiveConverter.Configuration Namespace
Provides configuration options for the [ObjectAsPrimitiveConverter](../ObjectAsPrimitiveConverter/ObjectAsPrimitiveConverter.md) JSON converter that handles objects as primitive types.

## Enums
| Name | Description |
| ---- | ----------- |
| [DetectDateTime](DetectDateTime.md) | Specifies the types of date and time representations to detect when converting text properties to primitive types. |
| [FloatFormat](FloatFormat.md) | Specifies the desired format for floating-point numbers when converting objects to primitive types. |
| [ObjectFormat](ObjectFormat.md) | Specifies the format to use when converting objects to primitive types. |
| [UnknownNumberFormat](UnknownNumberFormat.md) | Specifies the behavior when encountering a number format that is not recognized. |
